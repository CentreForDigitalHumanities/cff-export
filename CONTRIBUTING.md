# Contributing

If you have suggestions or requests for this application, feel free to open an issue on Github. Below is some additional information for development.

## Technical description

cff-export is a static website. It uses [citation.js](https://citation.js.org/) to convert citation data, and [vite](https://vite.dev/) as a bundler.

## Installation & development

Requires [NPM / Node.js](https://nodejs.org/en/download).

After cloning, install NPM dependencies with:

```sh
cd cff-export
npm install
```

Then you can start a development server with:

```sh
npm run dev
```
